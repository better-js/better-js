# text-align 文本对齐

::: tip 语法

属性：`text-align`

属性值：

- `left` ：靠左（默认值）
- `right` ：靠右
- `center` ：居中
- `justify` ：两端对齐

:::

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>文本对齐</title>
    <style>
      div {
        text-align: center;
      }
    </style>
  </head>
  <body>
    <div>Code Better</div>
  </body>
</html>
```
