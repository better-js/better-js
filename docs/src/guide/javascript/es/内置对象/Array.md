# Array实例方法

## Array.prototype.at()

> **`at(index)` 方法返回数组中指定索引对应的元素**

**参数**

- index



**返回值**

- 指定索引对应的值



**示例**

```js
const arr = [1, 2, 3, 4];

// 获取arr数组中下标为2的数据
console.log(arr.at(2)); // 3
```
