# 项目概述

## 目录结构

```
.
├─ docs                    # 项目根目录
│  └─ .vitepress           # 配置目录
│     ├─ cache             # 缓存目录
│     ├─ themeConfig       # 默认主题配置目录
│     └─ config.mts        # 配置文件
└─ src                     # 源目录
│  ├─ public               # 静态资源
│  └─ index.md
└─ ...

```

## 项目命令
```sh
# 安装依赖
yarn

# 启动项目
yarn dev

# 构建项目
yarn build
```
