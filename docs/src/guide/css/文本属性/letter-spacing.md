# letter-spacing 字母间距

- 字母间距（文字间距）：`letter-spacing`

- 属性值为像素（px），正值让间距增大，负值让间距缩小。

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>字母间距</title>
    <style>
      div {
        /* 字母间距 */
        letter-spacing: 20px;
      }
    </style>
  </head>
  <body>
    <div>Code Better</div>
  </body>
</html>
```
