# String

1. 字符串是个伪数组。
2. 通过 `单引号（''）`、`双引号（""）` 或 `反引号（``）`包裹的数据都叫字符串，单引号和双引号没有本质上的区别。
3. 在控制台输出 **<font color="black">黑色</font>**
4. 单引号或双引号一定要成对使用，支持互相嵌套，使用规则：**外双内单**或**外单内双**。
5. 必要时可以使用转义符 `\`，输出单引号或双引号。

#### 模版字符串

符号：` `` `

优点：支持换行，使开发更高效。

```js
//内容拼接变量时，用 ${} 包住变量
document.write(`姓名：${name}，年龄${age}`)
```

#### 转义字符

| 转义符 | 解释说明                 |
| ------ | ------------------------ |
| \n     | 换行符，n是newline的意思 |
| \ \    | 斜杠 \                   |
| \\'    | '单引号                  |
| \\"    | "双引号                  |
| \t     | tab缩进                  |
| \b     | 空格，b是blank的意思     |
