# M端实战

这是一个从零到一搭建 Vue3 后台管理系统的教程

## 代码规范

| 工具        | 说明                   |
| ----------- | ---------------------- |
| eslint      | JavaScript代码检测工具 |
| stylelint   | css样式检查工具        |
| prettier    | 代码格式化工具         |
| husky       | commit拦截             |
| commitlint  | 统一提交规范           |
| preinstall  | 统一包管理工具         |



## 环境准备

- node
- pnpm
