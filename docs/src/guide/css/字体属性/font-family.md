# font-family 字体族

::: tip 语法

属性名：`font-family`

```css {2}
.code-better {
  font-family: "宋体", "楷体";
}
```

:::

::: warning 注意

1. windows系统中，默认字体为 `微软雅黑`。

2. 使用字体的英文名字兼容性更好，如果字体名包含空格，必须使用引号包裹起来。

3. 可以设置多个字体，按照从左到右的顺序逐个查找，找到就用，没有找到就使用后面的，且通常在最后面写上`serif` （衬线字体）或 `sans-serif` （非衬线字体）
