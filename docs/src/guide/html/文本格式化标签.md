# 文本格式化标签  

文本格式化标签用于在 HTML 文档中设置文本的样式和格式，以增强文本的可读性和视觉效果。

| 标签名      | 标签名 | 效果 |
| ---------- | ------------------------ | ------------------------ |
| `<b></b>` | `<strong></strong>`      | <strong>文本加粗</strong> |
| `<i></i>` | `<em></em>`              | <em>文本斜体</em> |
| `<u></u>`  | `<ins></ins>`            | <ins>下划线</ins> |
| `<s></s>`  | `<del></del>`            | <del>删除</del> |

推荐使用 `strong`、`em`、`ins`、`del` 标签，这些标签含有突出重要性的强调语境。
