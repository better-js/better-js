# word-spacing 单词间距

- 单词间距：`word-spacing`（靠识别空格设置间距）

- 字母间距（文字间距）：`letter-spacing`

- 属性值为像素（px），正值让间距增大，负值让间距缩小。

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>单词间距</title>
    <style>
      div {
        /* 单词间距 */
        word-spacing: 20px;
      }
    </style>
  </head>
  <body>
    <div>Code Better</div>
  </body>
</html>
```
